## 条纹图案
### 目标：纯白变条纹
### 实现：
1. 颜色值周期性变化
2. 修改bootpack.c
* 书上实现 使用**与**运算
```c
for (i = 0xa0000; i <= 0xaffff; i++) {
    write_mem8(i, i & 0x0f);  // AND
}
```
note: cpu处理数值数据（加减乘除，改变数值大小）和图形数据（位运算 & ｜ ^ ~ << >> >>>，改变01排列方式）

* 其他实现 使用模运算
```c
for (i = 0xa0000; i <= 0xaffff; i++) {
    write_mem8(i, i % 0x10);  // cdq idiv ecx
}
```
note: 对数值`2^n`的模运算等价于对`2^n-1`的与运算
